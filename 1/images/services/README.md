# Imágenes de Servicios

Esta carpeta contiene las imágenes para cada servicio de automatización.

## Archivos disponibles

- `lighting.jpg` - Automatización de iluminación
- `blinds.jpg` - Control de cortinas inteligentes
- `security.jpg` - Seguridad y videovigilancia
- `smart-lock.jpg` - Cerraduras inteligentes
- `climate.jpg` - Climatización automatizada
- `audio.jpg` - Audio multiroom
- `voice.jpg` - Control por voz
- `integration.jpg` - Alexa / Google Home

## Recomendaciones

- **Formato**: JPG, PNG o WebP
- **Dimensiones recomendadas**: 800x600px o 1200x900px
- **Tamaño máximo**: 500KB por imagen
- **Aspect ratio**: 4:3 o 16:9

## Cómo reemplazar las imágenes

1. Prepara tus imágenes con los nombres exactos listados arriba
2. Asegúrate de que tengan buena calidad y resolución
3. Reemplaza los archivos placeholder en esta carpeta
4. Las imágenes se actualizarán automáticamente en el sitio

## Optimización

Para optimizar las imágenes antes de usarlas, puedes usar:
- [TinyPNG](https://tinypng.com/) - Compresión sin pérdida de calidad
- [Squoosh](https://squoosh.app/) - Herramienta de Google para optimizar imágenes
